## Pokemon Test

#For Android

To run :
$ yarn install
$ yarn android

To build :
$ cd android
$ ./gradlew assembleRelease

#For iOS

To run :
$ yarn install
$ cd ios
$ pod install
$ yarn ios (or) run from xcode

To build :

- Open xcode
- Choose device as Any iOS device arm 64
- Product > Clean build folder
- Archieve
